## 一、核心业务规则（修订版）

### 1.1 ETC激活后售卖会员

**ETC激活成功后，在激活成功页引导用户购买会员。**

- 用户ETC申办时通过互联网系统完成通行费签约（选择签约渠道：支付宝/微信/银联）
- **签约渠道需判断是否支持购买会员**，不支持则无法申办会员
- 用户收到ETC卡签后进行硬件激活，互联网系统通知营销系统激活成功
- **ETC激活成功页面**展示会员产品，引导用户购买
- 用户选择会员产品 → 同意会员服务协议 → **二次签约（会员签约，通过互联网系统→签约渠道）**
- 会员签约成功 → 营销系统发送扣费指令给ETC记账系统 → ETC记账系统通过签约渠道扣费 → 扣费成功 → 开通会员试用期（61天）
- 会员签约成功后进入**审核流程**，审核通过后正式生效
- 审核失败则订单取消，通知用户
- **三种产品扣费模式不同：**
  - 年卡：签约后一次性扣费¥138
  - 月卡：签约后扣首月¥16.8，后续每月自动扣费
  - 次卡：签约后不扣费，每次通行扣¥1/次，月封顶¥20
- 扣费成功：开通会员（61天免费体验期）→ 分配权益 → 通知粤运
- 扣费失败：ETC记账系统返回失败结果，不创建会员，记录失败，通知用户重新操作
- **用户可选择不购买会员**，跳过会员购买，仅使用基础ETC功能

### 1.2 一车一会员

**校验时机**：在用户选择会员产品时，校验车牌是否已有生效中的会员，有则不允许购买。

### 1.3 免费体验期

- 新会员开通后享受**61天**免费体验期（状态为TRIAL）
- 体验期内可随时取消，不产生费用
- **体验期到期前3天/1天**发送到期提醒通知
- 体验期满未取消，系统自动从签约渠道扣取会员费（**轮扣机制**：首次扣费失败后每日重试，连续3天失败自动取消）
- 扣费成功后状态变为ACTIVE，同时取消体验期权益、分配正式权益

### 1.4 会员产品类型

| 类型 | 说明 | 签约扣费 | 后续扣费 |
|------|------|---------|---------|
| 年卡（YEARLY） | 年度会员 | 一次性扣费¥138 | 到期续费¥138/年 |
| 月卡（MONTHLY） | 月度会员 | 扣首月¥16.8 | 每月自动扣费¥16.8 |
| 次卡（PER_USE） | 按次计费 | 不扣费 | 每次通行¥1/次，月封顶¥20 |

### 1.5 ETC维保期

- ETC设备有维保期，由**综合服务系统**管理
- 会员的维保到期时间（warrantyEndDate）来自综合服务系统
- 维保期内设备故障可享受"只换不修"权益
- 维保期外设备故障需用户自行承担

1. 会员售卖从"ETC申办时"改为"ETC激活成功后"

### 2.4 扣费引擎流程

#### 扣费校验

每次扣费前校验**会员状态**（非产品状态）：
- 会员状态为TRIAL/ACTIVE才允许扣费
- 会员状态为PENDING_CANCEL/EXPIRED/CANCELLED则拒绝扣费

## 三、外部系统对接

### 3.1 互联网系统（内部）

| 场景 | 方向 | 说明 |
|------|------|------|
| 通行费签约 | 发送 | ETC申办时完成通行费签约（→签约渠道） |
| 会员二次签约 | 发送 | 用户购买会员时完成会员签约（→签约渠道） |
| 签约结果回调 | 接收 | 签约成功/失败 |

| 扣费 | 接收 | 从签约渠道扣费 |

## 五、异常处理机制

| 场景 | 处理方式 |
|------|---------|
| 权益分配失败 | 记录日志，自动重试3次，仍失败则人工介入 |
| 粤运通知失败 | 记录日志，异步重试（指数退避），最多重试5次 |
| ETC记账系统通知失败 | 记录日志，异步重试，最多重试5次 |
| 扣费失败 | 记录失败日志，通知用户，每日轮扣重试 |
| 连续3天扣费失败 | 自动取消会员，取消权益，通知粤运+ETC记账系统 |
| 签约失败 | 提示用户重新操作 |
| 用户不购买会员 | 正常使用基础ETC功能，后续可从首页/会员页购买 |
